## Tech Stack

- **Framework:** Next.js 15.5.4 with App Router
- **Language:** TypeScript
- **Styling:** Tailwind CSS 4.0
- **Syntax Highlighting:** Prism.js
- **AI:** Google Gemini API
- **Build Tool:** Turbopack
